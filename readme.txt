# Proyecto VolviendoACasita

## Descripción

Bienvenido al repositorio del proyecto VolviendoACasita.

Volvien‌do a Casita es una plataforma comunitaria diseñada para facilitar la comunicación y colaboración entre personas que han perdido o encontrado mascotas. Con el objetivo de aumentar las posibilidades de reunir a las mascotas perdidas con sus dueños, la plataforma ofrece diversas funcionalidades clave:

Registro de Mascotas Perdidas y Encontradas: Los usuarios pueden registrar mascotas perdidas o encontradas, así como asociar sus propias mascotas al sistema para una mejor gestión y comunicación directa en caso de encontrar una mascota perdida.
Notificaciones y Búsqueda de Mascotas: El sistema envía notificaciones a usuarios cercanos sobre mascotas perdidas o encontradas en su área, fomentando una búsqueda más activa y colaborativa.
Gestión de Solicitudes: Los usuarios pueden gestionar las solicitudes recibidas, revisar mensajes, aceptar o rechazar solicitudes, y comunicarse con solicitantes para obtener más información.
Envío de Correos Electrónicos: El sistema envía correos electrónicos en varias situaciones, como cuando se envía una solicitud para reclamar una mascota o cuando un formulario ha sido revisado y aprobado.
Gestión de Imágenes de Mascotas: Los usuarios pueden subir imágenes de mascotas perdidas o encontradas, facilitando su identificación por otros usuarios y ayudando en el proceso de contacto y localización.
Historial y Seguimiento: Los usuarios pueden ver el historial de mascotas perdidas o encontradas que han reportado, así como el historial de solicitudes enviadas y recibidas, permitiendo un seguimiento efectivo de los informes de mascotas.
Volvien‌do a Casita ofrece un sistema integral que combina estas funcionalidades para ayudar a los usuarios a informar y buscar mascotas perdidas o encontradas, facilitando su localización y el contacto con sus dueños.
